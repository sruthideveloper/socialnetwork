# Social Network API

## Description
A backend web application built using Django that simulates a social networking platform. 
It allows users to register, create posts, follow users, and interact with content.

## Features
- User authentication (login/register)
- Create, update, delete posts
- Follow/unfollow users
- REST API endpoints

## Tech Stack
- Python
- Django
- SQLite
- REST APIs

## Future Improvements
- JWT Authentication
- React frontend integration
- Deployment on AWS

## How to Run
pip install -r requirements.txt
python manage.py runserver
